# 📘 AI Smart Study Assistant

An AI-powered web application that helps students study smarter by extracting text from PDF notes, generating summaries, creating quizzes, searching notes, and providing voice reading.

---

## 🚀 Features

- 📄 Upload PDF Notes
- 📚 Extract Text from PDF
- 🔍 Search Notes
- 🤖 AI Summary Generation
- 🎯 AI Quiz Generation
- 📊 PDF Dashboard
- 🌙 Dark Mode
- 📋 Copy Notes
- 🔊 Read Aloud
- ⏹ Stop Reading
- 📥 Download Summary
- 📱 Responsive UI

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Bootstrap
- PyMuPDF
- Gemini AI API
